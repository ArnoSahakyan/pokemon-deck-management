## Features

- **Drag and Drop**: Easily drag Pok√©mon cards from the new card stack and drop them into custom decks.
- **Pok√©mon Details**: Click on a card to view detailed information about the Pok√©mon, including abilities, weight, height, and base experience.

- **Reset Decks**: Clear all decks and start fresh with organizing your Pok√©mon cards.

## Installation

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/ArnoSahakyan/pokemon-deck-management
   cd pokemon-deck-manager
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```

4. **Open the app**:
   Visit `http://localhost:3000` in your browser.

## Usage

1. **New Card Stack**:
   - The top-left section displays a stack of new Pok√©mon cards.
   - Drag a card from the stack and drop it into one of the decks at the bottom.

2. **View Pok√©mon Details**:
   - Click on any card to view detailed information about the Pok√©mon in the center of the screen.

3. **Manage Decks**:
   - Create and organize your Pok√©mon decks by dragging and dropping cards between decks.

4. **Reset Decks**:
   - Click the **Reset** button to clear all decks and fetch a new set of Pok√©mon cards.

5. **Shuffle Cards**:
   - Click the **Shuffle** button to fetch a new set of random Pok√©mon cards for the new card stack.
